# Known Faces Folder

Add your reference face photos here.

## Guidelines:
- One face per image
- Clear, front-facing photos preferred
- Filename (without extension) will be the person's name
- Supported formats: .jpg, .jpeg, .png, .bmp

## Example:
```
known_faces/
├── john_smith.jpg
├── jane_doe.jpg
└── alice_johnson.jpg
```

## Tips for best results:
1. Good lighting
2. Face clearly visible
3. Minimal background clutter
4. Front-facing or slight angle (< 30°)
5. Resolution: at least 300x300 pixels
